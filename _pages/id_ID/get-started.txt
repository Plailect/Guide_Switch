### Bacaan yang Perlu Dibaca

Versi perangkat yang berbeda akan memerlukan langkah yang berbeda untuk mencapai tujuan akhir Homebrew. Halaman ini akan membantu Anda untuk memulai sesuai dengan perangkat Anda.

Sebelum memulai, pengguna Windows harus mengaktifkan opsi untuk menampilkan ekstensi file menggunakan [File Extensions (Windows)](file-extensions-(windows))!
{: .notice--info}

Sebelum memulai, Anda mungkin ingin memeriksa kartu SD Anda karena kesalahan menggunakan [H2testw (Windows)](h2testw-(windows)), [F3 (Linux)](f3-(linux)), or [F3X (Mac)](f3x-(mac))!
{: .notice--warning}

Pilih halaman yang sesuai untuk versi Anda dari tabel di bawah ini. Catatan bahwa "from" dan "to" merupakan bidang iklusif. Ini berarti bahwa, sebagai contoh, versi "dari 1.0.0 to 2.3.0" mencakup versi 1.0.0, 2.3.0, dan semua versi di antaranya.

Versi perangkat Anda dapat ditemukan di bawah menu System Update di tab System pada aplikasi Settings.

![]({{ "/images/screenshots/system-version.png" | absolute_url }})
{: .notice--info}

{% capture notice %}

Perhatikan bahwa perangkat mungkin tidak menampilkan versi sistem jika pembaruan yang diunduh menunggu instalasi.

Jika demikian halnya pada perangkat Anda, Anda dapat melihat versi Anda dari Mode Pemulihan / Pemeliharaan dengan menyalakan perangkat Anda sambil menahan (Power) + (Volume Atas) + (Volume Bawah).

{% endcapture %}
